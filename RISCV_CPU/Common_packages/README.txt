Here are the packages used.

rv_esquirol defines a new type of array and a new function to fill tables like ROM_Memory or InstrMemory.
rv_esquirol_controlunit_image is a table of 256 values. The ROM_Memory is filled with this table. This file comes from the xml file rvucrom.
rv_esquirol_application_image is a table of 256 values too. The InstrMemory is filled with this table. This file comes from the hex file test1er.
rv_esquirol2 defines new types for the SlaveDecodeur.
